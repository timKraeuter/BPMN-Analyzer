Temporal logic model checking is executed in groove. Thus the groove syntax for temporal logic has to be used.
  • A for all future paths (CTL)
  • E for some paths (CTL)
  • U for until
  • X for next
  • W for weak until
  • G for always (globally)
  • F for finally
  • Boolean operators: &, |, !, ->, <-, <->, true and false.
A full description can be found in chapter 4.2 of the groove user manual available here.
