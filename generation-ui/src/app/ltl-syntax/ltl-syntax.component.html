LTL model checking is executed in groove. Thus the groove syntax for LTL has to be used.
  • U for until
  • X for next
  • W for weak until
  • G for always (globally)
  • F for finally
  • Boolean operators: &, |, !, ->, <-, <->, true and false.
A full description can be found in chapter 5.2 of the groove user manual available here.
